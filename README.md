## About Mahoorify

Mahoor is the name of one of the seven Persian musical instruments. Mahoorify is the online platform for audio streaming and media services. 

## Features

- Search Music
- Check New Weekly Releases
- Play Music
- Song Comments
- Create Playlist
- User authentication
- Genre Quick Search
